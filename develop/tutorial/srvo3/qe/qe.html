QE to wannier90 interface¶

pw2wan_srvo3.in

&inputpp 
   outdir = './'
   prefix = 'srvo3'
   seedname = 'srvo3'
   spin_component = 'none'
   write_mmn = .true.
   write_amn = .true.
   write_unk = .true.
   write_dmn = .true.
   wan_mode = 'standalone'
/

$ mpirun -np 4 pw2wan.x -in pw2wan_srvo3.in

Wannier90¶

Execute wannier90.x for the actual wannierization. The input file is the same as that for the pre-processing run.

$ wannier90.x srvo3

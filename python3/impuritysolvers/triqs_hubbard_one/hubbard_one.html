Hubbard-I approximation: TRIQS/hubbard-IÂ¶

This impurity solver implements the Hubbard-I approximation. For installation, follow https://triqs.github.io/triqs/1.4/applications/hubbardI/

Note

Be sure to use the same C++ compiler as the one used to build the TRIQS library.

To invoke the Hubbard-I solver, set parameters as

[impurity_solver]
name = TRIQS/hubbard-I

QE to wannier90 interface¶

pw2wan_srvo3.in

&inputpp 
   outdir = './'
   prefix = 'srvo3'
   seedname = 'srvo3'
   spin_component = 'none'
   write_mmn = .true.
   write_amn = .true.
   write_unk = .true.
   write_dmn = .true.
   wan_mode = 'standalone'
/

$ mpirun -np 4 pw2wan.x -in pw2wan_srvo3.in

Wannier90¶

Execute wannier90.x for the actual wannierization. The input file is the same as that for the preprocessing run.

$ wannier90 srvo3

DMFT calculation¶

srvo3.ini

[model]
U = 6.0
J = 1.0
lattice = wannier90
seedname = srvo3
nelec = 1.0
ncor = 1
cshell = [(2, 3)]
bvec=[(1.627091,0.0,0.0),(0.0,1.627091,0.0),(0.0,0.0,1.627091)]

[system]
nk0 = 8
nk1 = 8
nk2 = 8
beta = 10.0

[impurity_solver]
name = TRIQS/cthyb
n_cycles{int} = 10000

[control]
max_step = 10

[tool]
broadening = 0.1
nk_line = 50
nnode = 5
knode=[(G,0.0,0.0,0.0),(X,0.5,0.0,0.0),(M,0.5,0.5,0.0),(G,0.0,0.0,0.0),(R,0.5,0.5,0.5)]
omega_max =4.0
omega_min =-4.0
Nomega = 400

DMFT setup: dcore_pre¶

$ dcore_pre srvo3.ini

Running self-consistent DFT+DMFT : dcore¶

$ dcore srvo3.ini

Post-processing and data analysis: dcore_post¶

$ dcore_post srvo3.ini

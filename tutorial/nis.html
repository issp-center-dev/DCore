QE to wannier90 interface¶

pw2wan_nis.in

&inputpp 
   outdir = './'
   prefix = 'nis'
   seedname = 'nis'
   spin_component = 'none'
   write_mmn = .true.
   write_amn = .true.
   write_unk = .true.
   write_dmn = .true.
   wan_mode = 'standalone'
/

$ mpirun -np 4 pw2wan.x -in pw2wan_nis.in

Wannier90¶

Execute wannier90.x for the actual wannierization. The input file is the same as that for the preprocessing run.

$ wannier90 nis

DMFT setup: dcore_pre¶

$ pydmf_pre nis.ini

Running self-consistent DFT+DMFT : dcore¶

$ pydmf nis.ini

Post-processing and data analysis: dcore_post¶

$ pydmf_post nis.ini
$ gnuplot nis_akw.gp

“+” indicates the original band structure.

Output-file format¶

dcore_pre¶

This program generates seedname.h5 file. It has two groups, dft_input and Dcore. See DFTTools for the details of the data structure in dft_input group. The data included in Dcore group is list below:

Name Type Description
Umat numpy.complex The values of \(U^i_{\alpha\beta\gamma\delta}\), where \(i\) corresponds to the kind of correlated shell and \(\alpha, \beta, \gamma, \delta\) denote orbitals at each correlated shell.

dcore¶

This program generates seedname.out.h5 file. All data are stored in dmft_out group. The following list summarizes data structure in the dmft_out group:

Name Type Description
iterations Int The total number of iteration steps.
chemical_potential float The chemical potential at each iteration step.
G_iw Group The local Green’s function in imaginary-frequency domain.
Sigma_iw Group The local self-energy in imaginary-frequency domain.
Delta_iw Group The hybridization function (Weiss field) in the DMFT.
G0-log Group The non-interacting local Green’s function at each iteration step.
G-log Group The local Green’s function in imaginary frequencies at each iteration step.
Sigma-log Group The local self-energy in imaginary frequencies at each iteration step.
parameters Group All input parameters read from ini file.

See TRIQS manual for the data structure of the Green’s function and self-energy.
